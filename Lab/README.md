# OOP Lab (CSEG1044) — Experiments

This folder contains experiment-wise lab handouts:

- `notes.pdf` (student handout)
- `latex/notes.tex` (LaTeX source)

Canonical experiment list is in `admin/course-plan/OOP Lab Manual.pdf`.

## General instructions (from lab manual)

- Write and execute programs independently.
- Follow standard Java coding conventions.
- Each submission must include **source code**, **output**, and brief **observations**.
- Faculty may customize the problem scenario, but the experiment title/objective should remain within the syllabus scope.

Note: As per course policy, **viva/quiz/test questions are not included** in these student handouts. Faculty copies are stored in `private/Lab/`.
