- `Student-Guidelines/group-formation-and-project-approval.pdf`
- `Student-Guidelines/group-registration-and-project-approval-form.pdf`
- `Student-Guidelines/milestones-and-submission-checklist.pdf`
- `Student-Guidelines/common-project-evaluation-rubric.pdf`
- `Student-Guidelines/final-submission-pack.pdf`
- `Student-Guidelines/demo-and-evaluation-guidelines.pdf`
- `Student-Guidelines/github-repository-submission-workflow.pdf`

Then choose one project theme from:

- `Project-Themes/`

Unit 06 is intentionally organized as a project-guideline section rather than a lecture section. The primary student-facing material for Unit 06 is the project-guideline pack plus the project-theme folders.
