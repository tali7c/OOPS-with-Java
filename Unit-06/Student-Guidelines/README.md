# Unit 06 Student Guidelines

This folder contains the main student-facing documents for `Unit 06`.

Use the files in the following reading order.

## Recommended Reading Order

1. `unit-06-student-handbook.pdf`
   - Start here for the complete overview of Unit 06.
2. `unit-06-capstone-guidelines.pdf`
   - Read the core expectations for the capstone project.
3. `group-formation-and-project-approval.pdf`
   - Read the rules for group creation and project approval.
4. `group-registration-and-project-approval-form.pdf`
   - Use this while preparing the group details for teacher approval.
5. `milestones-and-submission-checklist.pdf`
   - Follow this to understand the deliverables and their sequence.
6. `common-project-evaluation-rubric.pdf`
   - Read this to understand how the project is likely to be evaluated.
7. `final-submission-pack.pdf`
   - Read this before final submission and demo preparation.
8. `demo-and-evaluation-guidelines.pdf`
   - Use this for final demo readiness and viva preparation.
9. `github-repository-submission-workflow.pdf`
   - Read this when your group is ready to work with the approved repository.

## Practical Use

- Read `unit-06-student-handbook.pdf` first.
- Do not create a Classroom repository before the teacher approves the project.
- After approval, use `github-repository-submission-workflow.pdf`.
- Keep your repository `README.md` complete with full group information.

## Related Material

After reading the files in this folder, go to:

- `../Project-Themes/`

Choose the project theme approved for your group and then follow its project description, phases, and phase-hint PDFs.
