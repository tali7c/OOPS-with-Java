# OOPS with Java (B.Tech CSE — Sem 4)

Repository for course materials for **Object Oriented Programming** (CSEG1044):
- Unit-wise lecture slides/notes
- Lab experiments + mini project

## Structure

Lectures live in:
```text
public/
  Unit-01/
    Lecture-01/
      latex/          # slides.tex + notes.tex (+ build artifacts)
      slides.pdf
      notes.pdf
  Unit-02/
    Lecture-01/
    ...
```

Labs live in:
```text
public/
  Lab/
    Experiment-01-02/
      notes.pdf
      latex/notes.tex
    ...
```

Miscellaneous notes live in:
```text
public/
  Miscellaneous/
    Why-Interfaces/
      slides.pdf
      notes.pdf
```
